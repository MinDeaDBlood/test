# Stage 60. Проверка перевода окна Options

Исправлены строки окна Options, которые оставались на английском при выборе испанского, французского, итальянского или португальского языка.

Причина была смешанная. Для части строк ключи уже существовали, но значения в неанглийских INI файлах оставались английскими. Для другой части строк текст задавался только в Designer и не обновлялся в ApplyLocalizedText при смене языка в открытом окне.

Исправлены разделы Personalization, Logs, Startup options, Background processes, File associations, AppX display name options и AI search settings.

Технические названия, пути, DISM, AppX, Windows, WIM, ESD, ISO и примеры идентификаторов пакетов не переводились принудительно.
